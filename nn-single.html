Purpose & Context

This project extended the single artificial neuron into a full neural network capable of classifying handwritten digits from the MNIST dataset. Everything — weights, forward pass, loss calculation, backpropagation, and training loop — was written from scratch in a single NeuralNetwork class.

The work was structured across three days. Day 1 built and tested individual functions in isolation. Day 2 assembled them into the class. Day 3 trained three different architectures on real data and compared their behaviour.

Conceptual Explanation

MNIST images are 28×28 pixel grids, each flattened into a list of 784 values normalised to 0–1. The network outputs 10 probability values, one per digit class (0–9).

  • Hidden layers use ReLU activation
  • Output layer uses Softmax — converts raw values to a probability distribution summing to 1
  • Loss is cross-entropy: L = −log(p_correct)
  • Backpropagation computes gradients and nudges every weight in the direction that reduces loss

Training loop

def train(self, data, epochs=1, learning_rate=0.01):
    """
    Train on data. Print loss and accuracy per epoch.
    Returns (loss_history, accuracy_history).
    """
    # loss_history stores the loss for every single example across all epochs
    loss_history = []
    #accuracy_history stores the running accuracy at each point in training
    #each value is how many we got right divided by how many we have seen so far
    accuracy_history = []
    #for every epoch in number of epochs
    for epoch in range(epochs):
        #reset the loss and the correct count to measure new performance each time
        epoch_loss = 0.0
        epoch_correct = 0

        for label, pixels in data:
            #run forward pass to get layer outputs
            layer_outputs = self.forward(pixels)

            #calculate loss for this example
            loss = self._cross_entropy_loss(layer_outputs[-1][1], label)
            #add this example loss to running total for individual epoch
            epoch_loss += loss

            #record individual loss for plotting
            loss_history.append(loss)

            # check if prediction is correct
            correct = self.predict(pixels) == label
            if correct:
                epoch_correct += 1

            #record correct or incorrect for plotting
            #len(accuracy_history) tells me how many examples i have processed before this one
            # added the plus 1 because I kept getting zero division error for the first example start
            accuracy_history.append(epoch_correct / (len(accuracy_history) + 1))

            # update weights using backprop
            self._backprop(pixels, label, layer_outputs, learning_rate)

        avg_loss = epoch_loss / len(data)
        avg_acc = epoch_correct / len(data)
        print(f'Epoch {epoch + 1}: loss = {avg_loss:.4f}   accuracy = {avg_acc:.2%}')

    return loss_history, accuracy_history

Reflection

The large network reached 99.75% training accuracy but only 89% on the test set. With 535,818 parameters trained on only 800 examples, it memorised the training data rather than learning general patterns — overfitting. Like a student who memorises the exact questions but struggles when the phrasing changes.

The tiny network had too few parameters and underfitted. The medium network offered the best balance. That is the one I would choose for a real application — reasonable accuracy, fewer parameters, less risk of overfitting.

Fixing the backpropagation indentation bug was the most significant debugging challenge. The delta computation for the layer below was originally inside the wrong block, meaning gradients were not propagating correctly. Once the indentation was corrected, training converged as expected.

import random
import math
import matplotlib.pyplot as plt

class NeuralNetwork:

    def __init__(self, layer_sizes):
        self.weights = []
        self.biases  = []

        for i in range(len(layer_sizes) - 1):
            layer_weights = [
                [random.uniform(-0.1, 0.1) for _ in range(layer_sizes[i])]
                for _ in range(layer_sizes[i + 1])
            ]
            layer_biases = [0.0] * layer_sizes[i + 1]
            self.weights.append(layer_weights)
            self.biases.append(layer_biases)

    def __str__(self):
        sizes = [len(self.weights[0][0])]
        for layer in self.weights:
            sizes.append(len(layer))
        arch = ' -> '.join(map(str, sizes))
        total_params = sum(
            len(w) * len(w[0]) + len(b)
            for w, b in zip(self.weights, self.biases)
        )
        weight_count = sum(len(w[0]) for w in self.weights)
        bias_count   = len(self.biases[-1])
        return (f"NeuralNetwork: {arch} |----|"
                f"Activations: ReLU (hidden), Softmax (output)|----|"
                f"Total parameters: {total_params} |----| "
                f"Weight Count:{weight_count} |---| Bias Count {bias_count}")

    def _relu(self, x):
        """Return x if positive, otherwise 0."""
        return max(0.0, x)

    def _softmax(self, values):
        """Convert a list of values to a probability distribution."""
        max_val   = max(values)
        exp_values = [math.exp(value - max_val) for value in values]
        total      = sum(exp_values)
        return [value / total for value in exp_values]

    def _weighted_sum(self, weights, inputs, bias):
        """Dot product of weights and inputs, plus bias."""
        return sum(inp * weight for weight, inp in zip(weights, inputs)) + bias

    def _forward_layer(self, inputs, layer_idx, activation):
        """
        Compute the output of one layer.
        Returns (pre_activation, post_activation).
        """
        weights = self.weights[layer_idx]
        biases  = self.biases[layer_idx]
        #For every node in this layer, calculate its weighted sum. weights[node] is that nodes list of weights,
        #inputs is what came in,
        #biases[node] is that nodes bias. Collect all the results into the list called pre_activation
        pre_activation = [self._weighted_sum(weights[node], inputs, biases[node])
            for node in range(len(biases))] # len(biases) gives me the number of nodes in this layer. length directly = the number of nodes.
        # apply softmax across all pre activation values together
        if activation == 'softmax':
            post_activation = self._softmax(pre_activation)
        else:
            # apply relu to each value one at a time
            post_activation = [self._relu(value) for value in pre_activation]
        # return both raw and activated values
        return pre_activation, post_activation

    def forward(self, inputs):
        """
        Pass inputs through all layers.
        Returns a list of (pre, post) tuples, one per non-input layer.
        """
        layer_outputs = []
        current_data = inputs  # start with the raw inputs

        for index in range(len(self.weights)):
        #check if this is the last layer
            last_output = (index == len(self.weights) - 1)

        # last layer uses softmax, all other layers use relu
            activation = 'softmax' if last_output else 'relu'

        # pass current data through the layer
            pre_activation, post_activation = self._forward_layer(current_data, index , activation)

        # store the results
            layer_outputs.append((pre_activation, post_activation))

        # output of this layer becomes input to the next
            current_data = post_activation

        return layer_outputs

    def predict(self, inputs):
        """Return the predicted digit for a given input."""
        #call the forward Method
        layer_outputs = self.forward(inputs)
        #the final layers post activation values
        output_probs= layer_outputs[-1][1]
        #find the highest probability
        max_prob = max(output_probs)
        return output_probs.index(max_prob)

    def _cross_entropy_loss(self, output_probs, label):
        """Compute cross-entropy loss for one example."""
        return -1*math.log(output_probs[label] + 1e-12)

    def _backprop(self, inputs, label, layer_outputs, learning_rate):
        """
        Update weights using backpropagation.
        Bring across the provided backprop function,
        replacing net['weights'] with self.weights and so on.
        """
        n_layers = len(self.weights) # changed net to self

        # One-hot target: all zeros except a 1 at the correct class.
        target = [0.0] * 10
        target[label] = 1.0

        # Output layer gradient: cross-entropy + softmax simplify to output - target.
        output_probs = layer_outputs[-1][1]
        delta = [output_probs[i] - target[i] for i in range(len(output_probs))]

        for layer_idx in range(n_layers - 1, -1, -1):

            # The input to this layer is either the original input (layer 0)
            # or the post-activation output of the previous layer.
            if layer_idx == 0:
                layer_input = inputs
            else:
                layer_input = layer_outputs[layer_idx - 1][1]
        # Save current weights before updating.
        # We need the original weights to compute the gradient for the layer below.
            saved_weights = [row[:] for row in self.weights[layer_idx]] # net to self

        # Update each weight and bias.
            for j in range(len(self.weights[layer_idx])): # net to self
                for k in range(len(self.weights[layer_idx][j])):# net to self
                    # The gradient of the loss with respect to this weight is
                # delta[j] * layer_input[k].
                    self.weights[layer_idx][j][k] -= learning_rate * delta[j] * layer_input[k] # net to self
                self.biases[layer_idx][j] -= learning_rate * delta[j] # net to self

        # Compute delta for the layer below (chain rule).
            if layer_idx > 0:
                pre_activation = layer_outputs[layer_idx - 1][0]
                new_delta = []
                for k in range(len(layer_input)):
                # Sum contributions from all nodes in the current layer.
                    grad = sum(saved_weights[j][k] * delta[j]
                           for j in range(len(delta)))
                # ReLU's derivative: 1 if pre-activation was positive, else 0.
                    relu_deriv = 1.0 if pre_activation[k] > 0 else 0.0
                    new_delta.append(grad * relu_deriv)
                delta = new_delta

    def train(self, data, epochs=1, learning_rate=0.01):
        """
        Train on data. Print loss and accuracy per epoch.
        Returns (loss_history, accuracy_history).
        """
        loss_history = []
        accuracy_history = []
        for epoch in range(epochs):
            epoch_loss = 0.0
            epoch_correct = 0

            for label, pixels in data:
                layer_outputs = self.forward(pixels)
                loss = self._cross_entropy_loss(layer_outputs[-1][1], label)
                epoch_loss += loss
                loss_history.append(loss)
                correct = self.predict(pixels) == label
                if correct:
                    epoch_correct += 1
                accuracy_history.append(epoch_correct / (len(accuracy_history) + 1))
                self._backprop(pixels, label, layer_outputs, learning_rate)

            avg_loss = epoch_loss / len(data)
            avg_acc = epoch_correct / len(data)
            print(f'Epoch {epoch + 1}: loss = {avg_loss:.4f}   accuracy = {avg_acc:.2%}')

        return loss_history, accuracy_history
