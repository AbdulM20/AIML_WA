Neural Network

A fully modular neural network built with object-oriented design — swappable layers, activations, and output functions trained on MNIST.

Back to overview

Purpose & Context

Skills Demo 3 rebuilt the neural network from scratch with a new design principle: modularity. Instead of a single class holding everything, each component — layers, activations, output functions — became its own class with a shared interface. This made it possible to swap any part of the network without rewriting the training logic.

The project was completed across four parts: individual planning (Part A), group implementation (Part B), maths questions (Part C), and experiments (Part D). The experiments compared two architectures on MNIST.

Object-Oriented Design

The key design insight was that Model should not care what type of layer, activation, or output function it is working with — it just calls .forward() on whatever it has. This is polymorphism through inheritance.

  • Layer → InputLayer, HiddenLayer, OutputLayer — all share a forward(inputs) interface so Model can loop through them uniformly
  • Activation → ReLU, Sigmoid, Linear — any activation can be passed into a HiddenLayer at construction time
  • OutputFunction → Softmax, LinearOutput — the output layer’s behaviour is equally swappable

The Linear activation insight

Before writing any code, the planning phase asked: what would happen if you used Linear activation for every layer? The instinct written in the planning notebook: “Linear just passes the value through unchanged so stacking multiple linear layers is the same as having one — no matter how many layers I add the network would never be able to learn anything complex.”

Key Functions

Base Layer class and HiddenLayer

class Layer:
    """
    This is the base class for all Layers in this network.

    this defines the interface all the layers will follow meaning a forward method which
    accepts inputs, returns output. this class does nothing on its own.
    its purpose is that the input layer , hidden layer and output layer all share the same shape
    so that the model can loop through itself (self.layers) without caring which type it is.
    """
    def forward(self, inputs):
        """
        pass inputs through this layer and return the outputs,
        subclasses will override this
        """
        pass

class HiddenLayer(Layer):
    """ This should hold its own weights and biases as instance attributes
     so that the backpropagation can reach in and update them. here we can apply an activation
     function meaning Sigmoid, relu, linear after doing the weighted sums
     """
    def __init__(self,weights,biases,activation):
        self.weights = weights
        self.biases = biases
        self.activation = activation

    def forward(self, inputs):
        pre_activation_values = []
        """
        loop through each node in this layer
        zip pairs of each node's weight list with its corresponding bias
        so we don't need to index manually
        """
        for node_weights, node_bias in zip(self.weights, self.biases):
            """compute the weighted sum for this node
            zip pairs each individual weight with its corresponding input value
            weights * inputs multiplies them together
            sum() adds all those together
            then the bias is added to give the node the ability to fire even when all inputs are zero
            this gives the pre activation value for this node
            """
            weighted_sum = sum(weight * input_value for weight, input_value in zip(node_weights, inputs)) + node_bias
            # store this node's pre activation value
            pre_activation_values.append(weighted_sum)
        # store pre activation on the instance so the backprop can access them later
        self.pre_activation_values = pre_activation_values
        #applying the activation function to all pre activation values at once
        post_activation_values = self.activation.forward(pre_activation_values)
        return post_activation_values

Softmax output function

class Softmax(OutputFunction):
    """
    converts a list of raw values into a distribution.
    every value will be between 0 and 1 and they will all sum to 1.
    """
    def forward(self, inputs):
        # subtract the max value first so the numbers don't get too big.
        max_value = max(inputs)
        exponents = [math.exp(input_value - max_value) for input_value in inputs]
        total = sum(exponents)
        return [exponent / total for exponent in exponents]

make_network factory function

def make_network(layer_sizes, activation, output_function):
    """
    builds the layers needed to create a model.
    takes the sizes, activation and output function and returns a list of layers
    that can be passed straight into Model.
    """
    layers = [InputLayer()]

    for layer_index in range(1, len(layer_sizes)):
        num_inputs = layer_sizes[layer_index - 1]
        num_nodes  = layer_sizes[layer_index]

        # give each weight a small random value so no two nodes start the same
        # logic is if all weights were equal the network would never learn
        weights = [[random.uniform(-0.1, 0.1) for input_index in range(num_inputs)] for node_index in range(num_nodes)]

        # biases start at zero
        biases = [0.0] * num_nodes

        # the last layer gets the output function, everything else gets the activation
        if layer_index == len(layer_sizes) - 1:
            layers.append(OutputLayer(weights, biases, output_function))
        else:
            layers.append(HiddenLayer(weights, biases, activation))

    return layers

Reflection

Model B performed as predicted — better results and not significantly slower. Most of the learning happened in epochs 1 and 2, with diminishing returns from epoch 3 onwards. The loss dropped sharply then levelled off.

The biggest difference from the previous skills demo was the design approach. Having separate classes for layers and activations made the code much easier to reason about and experiment with. Testing a different activation meant passing a different object into make_network rather than rewriting the class.

The next step I would take is testing multiple hidden layers to see whether adding depth rather than just width improves performance on this dataset.

import random
import math

class Layer:
    """
    This is the base class for all Layers in this network.

    this defines the interface all the layers will follow meaning a forward method which
    accepts inputs, returns output. this class does nothing on its own.
    its purpose is that the input layer , hidden layer and output layer all share the same shape
    so that the model can loop through itself (self.layers) without caring which type it is.
    """
    def forward(self, inputs):
        """
        pass inputs through this layer and return the outputs,
        subclasses will override this
        """
        pass

class InputLayer(Layer):
    """
    this is the first layer in the network
    it will do nothing but accept inputs and pass them through unchanged meaning no weights,
    biases nor activations
    """
    def forward(self,inputs):
        return inputs.copy()
    """because when you are handing info the assumption is either it is modifiying
    the object your dealing with or the thing ou had it is"""

class HiddenLayer(Layer):
    """ This should hold its own weights and biases as instance attributes
     so that the backpropagation can reach in and update them. here we can apply an activation
     function meaning Sigmoid, relu, linear after doing the weighted sums
     """
    def __init__(self,weights,biases,activation):
        self.weights = weights
        self.biases = biases
        self.activation = activation

    def forward(self, inputs):
        """
        as we send it forward in from this layer we need to compute the weighted sum for each node
        then add the bias then apply the activation function

        we will also store the pre activation values for back propagation as learned in the
        previous skills demo these values are important
        we also need this to return a list of post activation values
        """
        pre_activation_values = []
        """
        loop through each node in this layer
        zip pairs of each node's weight list with its corresponding bias
        so we don't need to index manually
        """
        for node_weights, node_bias in zip(self.weights, self.biases):
            """compute the weighted sum for this node
            zip pairs each individual weight with its corresponding input value
            weights * inputs multiplies them together
            sum() adds all those together
            then the bias is added to give the node the ability to fire even when all inputs are zero
            this gives the pre activation value for this node
            """
            weighted_sum = sum(weight * input_value for weight, input_value in zip(node_weights, inputs)) + node_bias
            # store this node's pre activation value
            pre_activation_values.append(weighted_sum)
        # store pre activation on the instance so the backprop can access them later
        self.pre_activation_values = pre_activation_values
        #applying the activation function to all pre activation values at once
        post_activation_values = self.activation.forward(pre_activation_values)
        return post_activation_values

class OutputLayer(Layer):
    """ the final layer in the network
    like hidden layer it holds weights and biases as instance attributes.
    instead of an activation function it holds an output function
    meaning softmax or linear which produces the final predictions.
    """
    def __init__(self, weights, biases, output_function):
        self.weights         = weights
        self.biases          = biases
        self.output_function = output_function

    def forward(self, inputs):
        """
        compute weighted sums for all output nodes, then apply the output function.
        stores pre_activation_values for backpropagation.
        """
        pre_activation_values = []
        # same weighted sum logic as hidden layer
        for node_weights, node_bias in zip(self.weights, self.biases):
            weighted_sum = sum (weight * input_value for weight, input_value in zip(node_weights, inputs)) + node_bias
            pre_activation_values.append(weighted_sum)
        self.pre_activation_values = pre_activation_values
        #apply the output function to all pre-activation values at once
        #this gives us the final predictions of the network
        output_values = self.output_function.forward(pre_activation_values)
        return output_values

class Activation:
    """
    base class for all activation functions in this network.
    defines the system all activation functions must follow meaning a forward method
    that accepts a list of pre activation values and returns a list of post-activation values.
    """
    def forward(self, inputs):
        pass

class Sigmoid(Activation):
    """
    applies the sigmoid function to each value individually.
    squashes any value into a range between 0 and 1.
    """
    def forward(self, inputs):
        # apply sigmoid to each value in the list
        return [1 / (1 + math.exp(-input_value)) for input_value in inputs]

class ReLU(Activation):
    """
    applies the RELU function to each value individually.
    returns the value if positive, zero if negative.
    """
    def forward(self, inputs):
        # if the value is positive pass it through, if negative return zero
        return [max(0, input_value) for input_value in inputs]

class Linear(Activation):
    """
    passes values through unchanged.
    used in experiments to demonstrate that a network with no non-linearity
    collapses to a single linear transformation regardless of depth.
    """
    def forward(self, inputs):
        # returning the values unchanged
        return inputs.copy()

class OutputFunction:
    """
    base class for all output functions in this network.
    """
    def forward(self, inputs):
        pass

class Softmax(OutputFunction):
    """
    converts a list of raw values into a distribution.
    every value will be between 0 and 1 and they will all sum to 1.
    """
    def forward(self, inputs):
        # subtract the max value first so the numbers don't get too big.
        max_value = max(inputs)
        exponents = [math.exp(input_value - max_value) for input_value in inputs]
        total = sum(exponents)
        return [exponent / total for exponent in exponents]

class LinearOutput(OutputFunction):
    """
    passes values through unchanged.
    this is the alternative output function for experiments.
    """
    def forward(self, inputs):
        # return values unchanged
        return inputs.copy()

class Model:
    """
    holds all the layers and is responsible for running the network.
    it handles training, prediction, and evaluation.
    """

    def __init__(self, layers):
        """
        layers: a list of Layer objects in order from input to output
        """
        self.layers  = layers
        self.history = {'loss': [], 'accuracy': []}

    def forward(self, inputs):
        """
        pass the inputs through every layer in order.
        the output of each layer becomes the input to the next.
        returns the final output of the network.
        """
        current_values = inputs

        # pass data through each layer in order
        for layer in self.layers:
            current_values = layer.forward(current_values)

        return current_values

    def predict(self, inputs):
        """
        run a forward pass and return the predicted class index.
        the predicted class is whichever output node has the highest value.
        """
        output_values = self.forward(inputs)

        #find the index of the highest value that is the predicted class
        return output_values.index(max(output_values))

    def evaluate(self, dataset):
        """
        run predict on every example in the dataset.
        return which were correct as a float between 0 and 1.
        """
        correct = 0

        for inputs, label in dataset:
            #bug test
            if isinstance(label, list):
                label = label.index(max(label))
            if self.predict(inputs) == label:
                correct += 1

        return correct / len(dataset)

    def describe(self):
        """
        prints the structure of the network and the total parameter count.
        weights and biases both count as parameters.
        """
        total_parameters = 0

        for layer in self.layers:
            # only hidden and output layers have parameters
            if isinstance(layer, (HiddenLayer, OutputLayer)):
                layer_parameters = sum(len(node_weights) for node_weights in layer.weights) + len(layer.biases)
                total_parameters += layer_parameters
                print(f'layer: {len(layer.biases)} nodes, {layer_parameters} parameters')

        print(f'total parameters: {total_parameters}')

    def fit(self, dataset, epochs, learning_rate):
        """
        train the network on the dataset for a given number of epochs.
        stores loss and accuracy per epoch in self.history.
        """
        for epoch in range(epochs):
            epoch_loss = 0
            correct = 0

            for inputs, label in dataset:
                #bug test
                if isinstance(label, list):
                    label = label.index(max(label))
                # run the forward pass once and collect inputs to each layer
                layer_inputs = [inputs]
                for layer in self.layers:
                    layer_inputs.append(layer.forward(layer_inputs[-1]))

                output_values = layer_inputs[-1]

                # calculate the loss for this example
                loss = -math.log(output_values[label] + 1e-12)
                epoch_loss += loss

                # check if the prediction was correct
                if output_values.index(max(output_values)) == label:
                    correct += 1

                # backward pass updates weights and biases
                self._backprop(label, layer_inputs, learning_rate)

            # store the average loss and accuracy for this epoch
            self.history['loss'].append(epoch_loss / len(dataset))
            self.history['accuracy'].append(correct / len(dataset))

            print(f'epoch {epoch + 1}: loss {epoch_loss / len(dataset):.4f}, accuracy {correct / len(dataset):.4f}')

    def _backprop(self, label, layer_inputs, learning_rate):
        """
        update weights and biases using backpropagation.
        works backwards through the layers computing deltas and applying updates.
        layer_inputs is collected during the forward pass in fit() and passed in directly.
        """
        # compute delta for the output layer
        output_values = layer_inputs[-1]
        delta = [output_values[node_index] - (1 if node_index == label else 0) for node_index in
                 range(len(output_values))]

        # work backwards through hidden and output layers
        for layer_index in range(len(self.layers) - 1, 0,-1):
            layer = self.layers[layer_index]

            if isinstance(layer, (HiddenLayer, OutputLayer)):
                layer_input =layer_inputs[layer_index]

                # save weights before updating needed to compute delta for the layer below
                saved_weights = [node_weights[:] for node_weights in layer.weights]

                # update weights and biases for each node in this layer
                for node_index in range(len(layer.weights)):
                    for weight_index in range(len(layer.weights[node_index])):
                        layer.weights[node_index][weight_index] -= learning_rate * delta[node_index] * layer_input[weight_index]
                    layer.biases[node_index] -= learning_rate * delta[node_index]

                # compute delta for the next layer down
                if layer_index > 1:
                    previous_layer = self.layers[layer_index - 1]
                    pre_activation = previous_layer.pre_activation_values
                    new_delta = []

                    for node_index in range(len(layer_input)):
                        grad = sum(saved_weights[j][node_index] * delta[j] for j in range(len(delta)))
                        relu_deriv = 1.0 if pre_activation[node_index] > 0 else 0.0
                        new_delta.append(grad * relu_deriv)

                    delta = new_delta

def make_network(layer_sizes, activation, output_function):
    """
    builds the layers needed to create a model.
    takes the sizes, activation and output function and returns a list of layers

    for layer_index in range(1, len(layer_sizes)):

        num_nodes  = layer_sizes[layer_index]

        # biases start at zero
